# Go-system-monitor
Легковесный монитор системных ресурсов Windows в реальном времени. Проект использует Go на бэкенде для сбора данных через WMI и системные вызовы, и Chart.js на фронтенде для визуализации.
# если захотите поменять фронт,он должен лежать в static
